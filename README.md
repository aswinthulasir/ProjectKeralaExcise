### Getting started
To run frontend
1. ``` cd frontend ```
2. ``` npm i ```
3. ``` npm run dev ```

To run backend
1. ```cd backend```
2. ```npm i```
3. ```node app.js or nodemon app.js```

   if nodemon is not working, install it globally [npm i nodemon -g]

   ### Requirement
   1. 8GB RAM or more
   2. Min 256 SSD
   3. Graphics card makes it run smooth
